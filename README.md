# Titanic Dataset Analysis

## Project Overview

This project analyzes the Titanic dataset using Python and Pandas as part of the June Month AI/ML task.

## Dataset Used

* Titanic Dataset

## Libraries Used

* Pandas
* Matplotlib

## Steps Performed

1. Loaded the dataset using Pandas
2. Performed data cleaning
3. Generated summary statistics
4. Created visualizations
5. Extracted insights from the data

## Visualizations Created

* Survival Count Graph
* Age Distribution Histogram
* Passenger Class Distribution Graph
