# Titanic Survival Prediction using Machine Learning

## Project Overview

This project analyzes the Titanic dataset and builds multiple Machine Learning models to predict passenger survival. The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, performance evaluation, and comparison of different Machine Learning algorithms.

This project was completed as part of the June AI/ML Domain Task.

---

## Dataset Used

* Titanic Dataset

---

## Technologies and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

---

## Project Workflow

1. Loaded the Titanic dataset
2. Cleaned and preprocessed the data
3. Generated summary statistics
4. Created data visualizations
5. Prepared features (X) and target (y)
6. Split the dataset into training and testing sets
7. Trained multiple Machine Learning models
8. Evaluated model performance
9. Compared model accuracies
10. Identified the best-performing model

---

## Machine Learning Models Implemented

* K-Nearest Neighbors (KNN)
* Decision Tree
* Logistic Regression
* Random Forest

---

## Model Performance

| Model                     |   Accuracy |
| ------------------------- | ---------: |
| Random Forest             | **88.55%** |
| Logistic Regression       | **84.35%** |
| Decision Tree             | **80.53%** |
| K-Nearest Neighbors (KNN) | **75.95%** |

---

## Best Performing Model

Random Forest achieved the highest accuracy of **88.55%**, making it the best-performing model for predicting passenger survival in this project.
